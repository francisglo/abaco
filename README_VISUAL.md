## 🚀 CÓMO EMPEZAR

```bash
# 1. Navega a la carpeta del proyecto
cd C:\Users\Usuario\OneDrive\Escritorio\Abaco

# 2. Instala dependencias
npm install

# 3. Inicia los servidores
npm run dev              # Vite en :5174
npm run mock:server     # JSON-server en :4000

# 4. Abre en navegador
http://localhost:5174

# 5. ¡Explora!
- Home: Todos los algoritmos integrados
- Control: Monitoreo en tiempo real
- Optimization: Parámetros ajustables
```
